အခုခေတ်မှာ လူတယောက်တည်းတင် ဖုန်း၊ ကွန်ပျူတာ၊ Smart Watch ကစလို့ အိမ်က Smart TV တွေအထိ အင်တာနက်ချိတ်တဲ့ စက်တွေ အများကြီး သုံးလာကြတာ သတိထားမိကြလား။

ကမ္ဘာပေါ်မှာ လူဦးရေ ၈ ဘီလီယံလောက် ရှိပေမယ့် အင်တာနက် ချိတ်ဆက်အသုံးပြုနေတဲ့ စက်တွေက အဲဒီထက် အများကြီး ပိုများနေပါပြီ။ ပြီးခဲ့တဲ့ ဆောင်းပါး မှာ IP Address ဆိုတာ အင်တာနက်ပေါ်က "အိမ်လိပ်စာ" လို့ ပြောခဲ့တာ မှတ်မိကြဦးမယ် ထင်ပါတယ်။ အဲဒီ လိပ်စာတွေ ကုန်သွားရင် ဘယ်လိုလုပ်ကြမလဲ။

အဲ့ဒီ ပြဿနာကို ဖြေရှင်းဖို့ ပေါ်လာတာကတော့ ဒီနေ့ လေ့လာမယ့် IPv6 ပါပဲ။ စာမေးပွဲတွေမှာ ခဏခဏ မေးလေ့ရှိတဲ့ IPv4 နဲ့ IPv6 ဘာတွေ ကွာခြားလဲ၊ ဘာကြောင့် v6 ကို ပြောင်းသုံးနေရတာလဲ ဆိုတာကို အတူတူ ကြည့်လိုက်ရအောင်။

## 🧐 IPv4 နဲ့ IPv6 ဘာတွေ ကွာခြားလဲ?

### IPv4 (Internet Protocol version 4)

၁၉၈၀ ဝန်းကျင်လောက်ကတည်းက စသုံးခဲ့တဲ့ စနစ်ဟောင်းလေးပါ။  
Format အနေနဲ့ ဂဏန်းတွေချည်းပဲ သုံးပါတယ် (Decimal format)။  
အပိုင်း ၄ ပိုင်းကို အစက် (Dot) လေးတွေနဲ့ ပိုင်းခြားထားပါတယ်။  
( ဥပမာ: `192.168.1.1` )  

32-bit စနစ်ဖြစ်လို့ လိပ်စာပေါင်း ၄.၃ ဘီလီယံ (4.3 Billion) လောက်ပဲ ထုတ်ပေးနိုင်ပါတယ်။

---

### IPv6 (Internet Protocol version 6)

IPv4 လိပ်စာတွေ မလောက်တော့လို့ အသစ်ထပ်မံ တီထွင်လိုက်တဲ့ စနစ်သစ်ပါ။  
Format အနေနဲ့ ဂဏန်းရော၊ အင်္ဂလိပ် အက္ခရာ (A ကနေ F အထိ) ပါ ရောသုံးပါတယ် (Hexadecimal format)။  

အပိုင်း ၈ ပိုင်းကို အစက်နှစ်လုံး (Colon) လေးတွေနဲ့ ပိုင်းခြားထားပါတယ်။  
ဥပမာ:  
`2001:0db8:85a3:0000:0000:8a2e:0370:7334`

128-bit စနစ်ဖြစ်လို့ လိပ်စာပေါင်း 340 Undecillion (၃၄၀ နောက်မှာ သုည ၃၆ လုံး) အထိ ထုတ်ပေးနိုင်ပါတယ်။  
(ကမ္ဘာပေါ်က သဲပွင့်တိုင်းကို IP Address ပေးရင်တောင် လောက်တယ်လို့ တင်စားကြပါတယ်)

---

## ❓ ဘာကြောင့် IPv6 ကို မဖြစ်မနေ ပြောင်းသုံးရတာလဲ?

📌 အဓိက အကြောင်းရင်းကတော့ IPv4 ရဲ့ ၄.၃ ဘီလီယံဆိုတဲ့ အရေအတွက်ဟာ အခုခေတ် Smart Devices တွေ (Internet of Things - IoT) အရမ်းများလာတဲ့အခါ လုံးဝ မလောက်တော့လို့ပါပဲ။ အင်တာနက် ကြီးထွားလာမှုကို ဆက်လက် ထိန်းထားနိုင်ဖို့ IPv6 က မရှိမဖြစ် လိုအပ်လာပါတယ်။

📌 IPv6 က Data ပို့တဲ့ လုပ်ငန်းစဉ်မှာ Header လို့ခေါ်တဲ့ အချက်အလက်ပိုင်းကို ပိုမို ရှင်းလင်းအောင် တည်ဆောက်ထားလို့ Router တွေက Data တွေကို ပိုမြန်မြန် ပို့ဆောင်ပေးနိုင်ပါတယ်။

📌 IPv6 မှာ IPSec လို့ခေါ်တဲ့ လုံခြုံရေးစနစ်ကို မွေးရာပါ အနေနဲ့ ထည့်သွင်းတည်ဆောက်ထားလို့ Data တွေ ကြားဖြတ် ခိုးယူခံရတာမျိုး မဖြစ်အောင် ပိုမို ကာကွယ်ပေးနိုင်ပါတယ်။

## 💡 Study Tips

- Bit အရေအတွက်ကို မှတ်ထားပါ။  
  စာမေးပွဲမှာ Multiple Choice အနေနဲ့ အရမ်း မေးလေ့ရှိပါတယ်။  
  IPv4 က 32-bit, IPv6 က 128-bit။

- Format ကို ယှဉ်ကြည့်ပါ။  
  IPv4 ဟာ Dot (.) သုံးပြီး Decimal ဂဏန်းတွေနဲ့ ဖွဲ့စည်းထားသလို  
  IPv6 ဟာ Colon (:) သုံးပြီး Hexadecimal တွေနဲ့ ဖွဲ့စည်းထားပါတယ်။

- IPv6 ကိုဘာကြောင့်အသုံးပြုတာလဲဆိုတဲ့ မေးခွန်းက  
  အတော်လေး မေးလေ့ရှိတဲ့ မေးခွန်းဖြစ်လို့  
  သေချာမှတ်ထားဖို့လိုပါတယ်။
